# 📊 Spotify Analysis - SQL
Analyzed Spotify music data using SQL to understand trends in songs, artists, and user preferences. Used queries to explore top tracks, most popular artists, and audio features such as popularity, duration, and release trends to gain insights into listening behavior and music patterns.


## 📌 Project Description
Analyzed Spotify music data using SQL to understand trends in songs, artists, and user preferences. Used queries to explore top tracks, most popular artists, and audio features such as popularity, duration, and release trends to gain insights into listening behavior and music patterns.

---

## 🎯 Objectives
-To analyze Spotify data and understand music trends
-To identify top songs, artists, and most popular tracks
-To study user preferences based on song popularity and features
-To extract insights using SQL queries like filtering, grouping, and aggregation

---

## 🛠️ Tools & Technologies
- SQL (MySQL)  
- MySQL Workbench / DBeaver (Database Tool)  
- GitHub  

---

## 📂 Dataset Description
- Database Name: ft_a
- Tables Name:
- Users
- Playlists
- Tracks

---

## 📈 Key Insights
- Identifies music trends like albums, tracks, and release patterns
- Shows user registration behavior (email domains & dates)
- Analyzes playlists and user engagement
- Understands track duration and listening patterns
